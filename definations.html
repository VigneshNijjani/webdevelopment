Element:
Combination of opening tag, closing tag and content
Single tags
Tags which are having only either opening tag or Closing tag Ex: hr, br, img, input
Paired tags
Tags which are having both opening and closing tags Ex: heading tags, p, pre
Text Formatting tags:
Formatting tags are used to change the appearance of text like making it bold, italic, underlined, small, big, marked, etc.
Attributes:
Attributes are one of the core component in html It will provide the additional information about an Html tag or element Attributes are always written in open tag Syntax: attribute name=”value”
Types of attributes:
1.Global attributes 2.Element specific attributes
Global Attributes:
These attributes are used for every html element Ex: id, class, style, title
Element specific attributes:
These attributes are used for particular html element Ex: src, alt
Marquee tag:
The marquee tag is used to create scrolling (moving) text or images inside a webpage.
Types of elements:
1.Inline elements 2.Block level elements
Inline elements:
• These elements take as much as width its necessary. • They did not start in new line • height and width are not affected except for img tag • margin and padding are applicable only left and right not top and bottom ex: button , a , img , formatting tags, span
Block level elements:
• They always start in a new line • They occupy 100% width • height and width are affected • Margin & padding → work in all directions (top, bottom, left, right).
Inline block elements:
• Behave like inline (stay in the same line) • But you can set width and height like a block element
Difference between Div and Span:
Difference between Div and Span: Use div for big blocks of content (layout/sections). Use span for small inline styling (highlight/word formatting).
HTML Lists:
A list in HTML is a way to group related items together in a structured format using special tags.
Types of Lists in HTML:
1.Ordered list 2.Un Ordered list 3.Definition/Description list
Ordered List:
• An ordered list displays items with numbers/letters/roman numerals. • For Ordered list we use ol tag
Unordered list:
• An unordered list displays items with bullets. • For unordered list we use ul tag
.Definition/Description list:
Used for terms and their descriptions. dl → Description list container dt → Description term dd → Description definition
